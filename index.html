Struttura e gestione dei database

Cos'è il DDL?

Il DDL (Data Definition Language) permette di creare e modificare la struttura dei database.

  • 🔹 CREATE DATABASE
  • 🔹 CREATE TABLE
  • 🔹 ALTER TABLE
  • 🔹 DROP TABLE

Gestione Database

Creazione:

CREATE DATABASE Scuola;

Eliminazione:

DROP DATABASE Scuola;

Creazione Tabelle

CREATE TABLE Studenti (
    ID INT PRIMARY KEY,
    Nome VARCHAR(50) NOT NULL,
    Eta INT CHECK (Eta >= 6)
);

Specifica Campi, Tipi e Vincoli nello stesso comando.

Modifica ed Eliminazione

Aggiungere una colonna:

ALTER TABLE Studenti ADD Email VARCHAR(100);

Eliminare la tabella:

DROP TABLE Studenti;

Modifiche Avanzate

-- Rimuovere una colonna
ALTER TABLE Studenti DROP COLUMN Email;

-- Aggiungere un vincolo complesso
ALTER TABLE Studenti 
ADD CONSTRAINT chk_eta 
CHECK (Eta >= 6 AND Eta <= 19);

Tipi di Dato

  • INT → Numeri interi
  • DECIMAL → Prezzi/Valute
  • VARCHAR → Testi e stringhe
  • DATE → Date (AAAA-MM-GG)
CREATE TABLE Prodotti (
  ID INT PRIMARY KEY,
  Prezzo DECIMAL(6,2),
  Disponibile BOOLEAN
);

Vincoli SQL (Constraints)

✅ NOT NULL: Obbligatorio

✅ UNIQUE: Non duplicabile

✅ PRIMARY KEY: ID Unico

✅ CHECK: Regola logica

CREATE TABLE Utenti (
  Email VARCHAR(50) UNIQUE,
  Eta INT CHECK (Eta >= 18)
);
